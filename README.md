🧪 化學實驗室數據分析系統 (Chemistry Lab Analysis)
這是一個自動化數據分析工具，能夠從實驗室資料庫中讀取樣本數據，並自動生成報表與圖表。

🚀 功能特色
資料庫串接：使用 Python (sqlite3) 連接 SQL 資料庫。
數據清洗：使用 Pandas 整理實驗數據。
自動報告：一鍵匯出 Excel (.xlsx) 格式清單。
視覺化：自動繪製「樣本質量分佈圖」長條圖。

🛠️ 使用技術
Python 3.10+
SQLite
Pandas
Matplotlib
